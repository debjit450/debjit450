### ⚡ [gitwit](https://github.com/debjit450/gitwit) — AI Git Commit Message Generator · ⭐ Open Source
`Node.js` `OpenAI API` `CLI` · `npm install -g gitwit`


- 📝 Generates [Conventional Commits](https://conventionalcommits.org/)-compliant messages from `git diff --cached`

- 🚀 Zero config, zero friction — works via `npx gitwit` or global install
- 🔒 Secure — your code never leaves your machine beyond the OpenAI API call
